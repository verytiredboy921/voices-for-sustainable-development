# Voices for Sustainable Development

## Overview

Voices for Sustainable Development is a student-driven awareness platform aligned with:

- SDG 11 – Sustainable Cities and Communities
- SDG 16 – Peace, Justice and Strong Institutions

The platform allows users to create, edit, and manage awareness-based articles that promote structured youth dialogue around sustainability and institutional transparency.

---

## Features

- Create awareness posts
- Edit existing posts
- Delete posts
- Category tagging (SDG 11 / SDG 16)
- Clean structured UI
- Responsive layout

---

## Tech Stack

- Node.js
- Express.js
- EJS (Embedded JavaScript Templates)
- CSS
- Git & GitHub

---

## How to Run Locally

1. Clone the repository
2. Run:

   npm install

3. Start the server:

   node index.js

4. Open:

   http://localhost:3000

---

## Purpose

This project was developed as a hackathon submission to demonstrate how structured digital platforms can empower youth participation in sustainable development discussions.
